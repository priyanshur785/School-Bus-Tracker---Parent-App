# 🚌 School Bus Tracker - Parent App

A mobile-first web application that allows parents to track their child’s school bus in real-time, customize notifications, and view bus history — built using **Next.js**, **Tailwind CSS**, and **Radix UI**, generated via **V0.dev**.

---

## 📱 Features

- 🔒 **Secure Login** for parents
- 📍 **Live Bus Tracking** with ETA and map view
- 🔔 **Custom Notification Settings**
- 👧 **Child Profile & Trip History**
- 📞 **Emergency Contact & Support Access**
- 🚐 (Optional) **Bus Details Page**

---

## 💻 Tech Stack

| Layer       | Tech Used                        |
|-------------|----------------------------------|
| Frontend    | Next.js 14 (App Router)          |
| Styling     | Tailwind CSS, Tailwind-animate   |
| UI Library  | Radix UI + ShadCN Components     |
| State Mgmt  | React Hook Form, Zod (Validation)|
| Charts      | Recharts                         |
| Icons       | Lucide React                     |
| Extras      | Date-fns, CmdK, Embla Carousel   |

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/your-username/school-bus-tracker.git
cd school-bus-tracker
```
### 2. Install dependencies
Using pnpm (recommended):

```bash
pnpm install
```
Or using npm:
```bash
npm install
```
### 3. Run the app
```bash
pnpm dev
```
or
```bash
npm run dev
Open http://localhost:3000 to view in browser.
```
